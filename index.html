Magento Certification Competition

I am giving away one voucher for a free Magento certification exam. I am hoping that by opening this up, others who are not planning on taking the test will also give away theirs, so there will potentially be more than one available in the future. For right now, there is only one, and I am giving it away.

What you have to do for it

I would like to see some blog posts for the Magento development community. I want everyone to write blog posts about why they choose to develop for Magento. It doesn't necessarily have to be persuasive, but I'd love to see something that can bring more people into Magento. If I know one thing, it's that every Magento shop on earth is hiring.

In this repo, add your blog post, preferrably as markdown, into the entries folder. You can see an example that I have there.

Rules

Every blog post needs to be posted somewhere publicly as well as added as a pull request to this github repo. The winner will be decided by votes from the community in this github repo, so if you're not participating in the contest, make sure to voice your opinion! A valid vote will be a comment on the pull request showing support. My voucher is only good until the end of the year, so I will need to have all of the blog posts in and ready for judging by November 30th. I will close out the PRs and notify the winner by December 3rd 11:59:59PM EST.

If you have a voucher too

Please contact me and I'll up the number of prizes for this. It'd be great if we got enough prizes for every blog post!
